How To Play

Tutorial how to play in your device

  • ANDROID

    1. Download and install the latest version of Growtopia from Google Play.
    2. Open Growtopia and log in to get your items updated.
    3. Download the hosts file below.
    4. Open Virtual Hosts and tap on Select Host File and select the file you just downloaded.
    5. Turn the switch on and log in to Growtopia.

    Download host Download Virtual Host

  • PC

    1. Open notepad as administrator
    2. Click on File and select Open
    3. Go to path C:\Windows\System32\drivers\etc and open Hosts file
    4. Add the server ip in thats file, Like this:
    15.235.166.218 growtopia1.com
    15.235.166.218 growtopia2.com
    5. Click File and select Save
    6. Now log in to Growtopia

    Copy ip address

  • IOS

    1. Go to the App Store and download/install Surge 5.
    2. Once installed. Open the application.
    3. One it opens. Click the Start or Setup Button on top right.
    4. Scroll down and find DNS. Choose Local Mapping.
    5. Click on Add New.
    6. Put growtopia1.com on Domain and Put [ 15.235.166.218 ] on Value.
    7. Click Done on top right
    8. Click on Add New.
    9. Put growtopia2.com on Domain and Put [ 15.235.166.218 ] on Value.
    10. Click Done on top right
    11. Done! Now you can open Growtopia
